# Global Chocolate Sales Analytics Dashboard 

## Project Overview
An interactive Power BI dashboard designed to analyze global sales performance, product trends, and regional distribution for a chocolate retailer.

## Key Insights & KPIs
* **Total Revenue:** $19.79M
* **Total Volume:** 540K Boxes sold.
* **Top Performer:** Australia leads in total sales ($3.6M).
* **Best Selling Product:** Smooth Sliky Salty ($1.12M).

## Visualizations Included
1. **Sales by Country:** A dual-axis chart comparing revenue vs. box volume.
2. **Temporal Trends:** Monthly sales performance showing peaks in February and June.
3. **Product Hierarchy:** A Treemap visualizing the market share of various chocolate types.
4. **Key Metrics:** High-level cards for Avg Order Value and Total Sales.

## Tools Used
* **Power BI Desktop:** Report authoring and DAX calculations.
* **Power Query:** Data cleaning and transformation.
